sketch绝好用的插件一：User Flows

　　终于做完了app，搞完了规范，本想喘口气歇歇，等等，发给开发之前，为了开发看的明白，看的透彻，看的舒爽，咱们还得搞个，，流程图……一两百个界面……哭…搞的头晕有木有。今天发现了一个神器：User Flows，童鞋们欢呼吧，雀跃吧，sketch简直良心大大的好哇，太特么贴心了！
　　废话不多说，来来搞起来

下载，安装

（怎么安装我就不说了，轻且浅的小朋友不在赘述之列）

官方下载地址https://abynim.github.io/UserFlows/

使用

　　需要定义两个画板之间的链接：选择一个图层和目标画板后执行定义链接命令 Plugins > User Flows > Create Link快捷键是 command + shift + k

此处注意：是选中一个图层和一个画板，不是一个图层和另外一个图层

添加条件

　　可执行 添加 / 编辑条件命令 Plugins > User Flows > Add/Edit Conditions或使用快捷键command + shift + D 然后放在合适的位置即可
编辑条件：

　　保存后，出现一小画板:

　　点击条件按钮和下一层次的画板做连接：

显示和隐藏链接：

　　隐藏连接线Plugins > User Flows > Hide Connections
　　所有快捷键：Plugins > User Flows > User Shortcuts可以自己设定快捷键

　　想改变连接线的颜色和粗细也是没问题：

生成流程图

　　定义所有链接后，执行生成流程图命令 Plugins > User Flows > Generate Flow Diagram 或使用快捷键 command + shift +F选择开始屏幕，输入流程的详细信息，然后单击生成

　　生成的文件是新建了一个页面，默认是_flows，是图片不可编辑。

注意：可自定义从哪张图片开始导出，但是没有定义在哪张结束，默认的是只导出有连接线的图，一连串的，突然断了，他就不导出了。所以想要这个流程图的童鞋，每张都不要断哦
